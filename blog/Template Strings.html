Template String

Template literal are strings literals allowing embedded expressions. You can use multi-line stings and string interpolation features with them. They were called "Template Strings" in prior editions of the ES2015 specification.

Template literals can contain placeholders. These are indicated be the dollar sign and curly braces (${expression}). The expressions in the placeholders and the text between the back-ticks (``) get passed to a function. The default function just concatenate the parts into a single string.
